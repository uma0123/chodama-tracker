# PS BANK — パチスロ貯メダル・貯玉まとめアプリ

パチンコ・スロットの貯玉/貯メダルを店舗別に記録し、換金額・推移グラフ・SNSシェアまで一気通貫で扱えるWebアプリです。

## 公開URL（スマホ／PCでそのまま使用可）

**https://uma0123.github.io/chodama-tracker/**

iPhoneのSafariで開いて「共有 → ホーム画面に追加」でアプリのように起動できます（オフライン対応）。

---

## このフォルダ（PS BANK）の中身

WebアプリやPWA（ホーム画面追加）の仕様で **ファイル名は固定** です（フォルダ名は変更可）。
それぞれの役割は下記の通り。

> 事業全体の構造は `会社案内.md` を参照（部署別フォルダで運用）。

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（HTML+CSS+JS全部入り）。これがメイン |
| `manifest.json` | PWA設定（アプリ名・アイコン・テーマカラー） |
| `sw.js` | Service Worker（オフライン動作・キャッシュ） |
| `icon-180.png` | iPhoneホーム画面用アイコン（apple-touch-icon） |
| `icon-192.png` | Android/PWA用アイコン |
| `icon-512.png` | 大サイズアイコン（OS共有時など） |
| `pachibank-logo.png` | ロゴ画像（参照用ストック。アプリ内では未使用） |
| `README.md` | このファイル |
| `.gitignore` | Git管理除外設定（.DS_Store等） |
| `.git/` | Gitリポジトリ情報（編集禁止） |

## アプリの主な機能

- **店舗追加・削除**：法人（マルハン／キコーナ／ベラジオ等）も選択
- **スロット／パチンコの両方を1店舗で記録**（貸玉・交換率も店ごとに設定）
- **稼働モード**：使用・回収を入力 →「転記」で残高に反映
- **月別／日別グラフ**：スロット/パチンコ/合計の3本線、月チップで日別ドリルダウン
- **凡例トグル**：スロット／パチンコ／合計の表示・非表示を切替（シェア画像も反映）
- **SNSシェア**：今表示している内容を1080×1080のPNGにしてX/LINE/Instagram等に投稿
- **フィルタ**：店舗名検索／法人別／並び順／残高あり店舗のみ
- **お気に入り★**：リスト先頭にピン留め
- **データのエクスポート／インポート**（JSON）で端末間移行
- **PWA**：ホーム画面追加でアプリ化、オフライン動作

## 更新・公開の手順（自分用メモ）

ローカルで `index.html` を編集 → ターミナルで以下：

```
cd "~/Desktop/ギャンブル期待値/slot/PS BANK"
git add index.html
git commit -m "更新内容"
git push
```

※フォルダ名に全角＆とスペースを含むため、シェルでは必ずダブルクオートで囲む。

push後 1〜2分で公開URLに反映されます。

---

## データの保存場所と移行

- 入力データは **各ユーザー端末のブラウザのlocalStorage** に保存（サーバー送信なし）
- 端末／ブラウザを変えたい場合：旧側で「設定 → エクスポート」でJSONをコピー → 新側で「インポート」で貼り付け
